# Loan Default Risk Analysis

## Project Overview
This project analyzes loan applicant data to identify high-risk customers and understand factors affecting loan default.

---

## Tools & Technologies
- Python
- Pandas
- Jupyter Notebook
- Matplotlib
- Seaborn

---

## Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Loan Default Pattern Analysis
- Risk Segmentation
- Data Visualization

---

## Key Insights
- Customers with low credit history had higher default probability
- High loan amount increased financial risk
- Income and employment status influenced repayment behavior
